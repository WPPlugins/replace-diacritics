== Description ==

This plugin has two modes of transforming romanian diacritics: in HTML codes or standard letters.
It works on any text encoding type(Central European, Western European, Baltic, etc.)

== Installation ==

1. Upload `replace-diacritics.zip` and extract it to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to `Settings` menu and select your type of replacement
